A fun, interactive Valentine's Day web app where the "No" button runs away when you try to click it!

## Setup

1. **Update Node.js** (if needed):
   - Download the latest LTS version from [nodejs.org](https://nodejs.org/)
   - Current project requires Node.js 20.19.0+ or 22.12.0+

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the dev server**:
   ```bash
   npm run dev
   ```

4. **Open your browser** to the URL shown (typically http://localhost:5173)
